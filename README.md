# Matrix AI Bot
matrix ии бот!

чтобы запустить бота нужно:
1. npm install matrix-bot-sdk
2. скачать промпт или создать свой.
3. получить токен, инструкция - https://t2bot.io/docs/access_tokens/
4. node bot.ts - запуск бота.
